# Required Features
- Display items (use any json file with any structure; at least 6 items)

- Each item has a visible “Add to Cart” button

- Clicking “Add to Cart” updates the cart state (React useState)

- A button toggles showing/hiding the cart panel

# Optional Features
- Cart displays item name and price(10 points)
- Use localStorage so refreshing the page does not lose the cart.(10 points)
- Load item list using fetch from a JSON file or public API instead of hardcoding.(10 points)
- Clear-cart button(10 points)
- Display small thumbnail images of items inside the cart(10 points)
- “Empty cart” message when no items exist.(5 points)
- Basic styling to make the layout readable.(5 points)
